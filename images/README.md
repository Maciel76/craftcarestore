# Pasta de Imagens dos Perfumes

## Como adicionar imagens dos perfumes

### Formato dos nomes dos arquivos:

Os nomes dos arquivos de imagem devem seguir este padrão:

- Nome do perfume em letras minúsculas
- Espaços substituídos por hífens (-)
- Caracteres especiais removidos
- Extensão .jpg

### Exemplos:

| Nome do Perfume    | Nome do Arquivo          |
| ------------------ | ------------------------ |
| 212 VIP ROSE       | `212-vip-rose.jpg`       |
| GOOD GIRL          | `good-girl.jpg`          |
| CREED AVENTUS      | `creed-aventus.jpg`      |
| BACCARAT ROUGE 540 | `baccarat-rouge-540.jpg` |
| LA VIE EST BELLE   | `la-vie-est-belle.jpg`   |

### Especificações recomendadas:

- **Formato**: JPG ou JPEG
- **Dimensões**: 300x300px ou 400x400px (proporção quadrada)
- **Tamanho do arquivo**: Menor que 200KB para melhor performance
- **Qualidade**: Alta resolução, fundo branco ou transparente

### Imagem padrão:

O arquivo `default.jpg` será exibido quando a imagem específica do perfume não for encontrada.

### Passos para adicionar imagens:

1. Obtenha imagens de alta qualidade dos perfumes
2. Redimensione para 300x300px ou 400x400px
3. Renomeie seguindo o padrão acima
4. Salve na pasta `images/`
5. As imagens aparecerão automaticamente nos cards

### Dicas:

- Use imagens com fundo branco ou transparente para melhor visualização
- Mantenha a qualidade consistente entre todas as imagens
- Você pode usar sites como Remove.bg para remover fundos
- Considere usar Photoshop, GIMP ou Canva para edição

### Ferramentas úteis:

- **Remove.bg**: Remover fundo de imagens
- **TinyPNG**: Comprimir imagens sem perder qualidade
- **Canva**: Criar e editar imagens online
- **GIMP**: Editor de imagens gratuito
